# Near Expiry

## A multi-vendor e-commerce web frontend

This project has been built with only the static contents. This web frontend has full e-commerce UI developed with with HTML5, Tailwind CSS, Alpine.js.

### Tools and Libraries

-   HTML5
-   Tailwind CSS
-   Alpine.js
-   Owl Carousel
-   Bootstrap Icons
-   Animate CSS
